# dtg300_zombiegame
DTG 300 Zombie Survival Game

Written in python 2.7 with pygame 2.9.1. 

Controls:
 - Left/Right: Move
 - Space:      Jump
 - Z:          Fire
 - X:          Throw Grenade
 - 1:          Shotgun
 - 2:          Pistol
 - 3:          Machete
